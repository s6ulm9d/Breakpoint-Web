# BREAKPOINT - Web Interface

This is the official landing page and web interface for the BREAKPOINT (Elite Edition) security engine.

## Overview
Built with React 19, Vite, and Three.js, this project serves as:
1.  The marketing/landing page for the tool.
2.  The documentation hub for installation and usage.

## Setup & Development

### Prerequisites
- Node.js (v18+)
- npm

### Installation
1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

### Building for Production
To generate the static assets for deployment:
```bash
npm run build
```
The output will be in the `dist` directory.

## Project Structure
- `src/components`: React components for each section (Hero, Contact, Features).
- `src/canvas`: Three.js 3D background logic.
- `public`: Static assets.
